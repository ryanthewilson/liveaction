## Project Overview

This is a React application built with Vite and styled with Tailwind CSS. It's a chat interface that allows users to interact with up to five AI agents.

## Getting Started

1.  **Install dependencies:** Run `npm install` to install all the necessary packages.
2.  **Start the development server:** Run `npm run dev` to start the Vite development server.
3.  **Open the application:** Open your browser and navigate to the local development URL provided by Vite.

## Coding Conventions

*   Follow standard React and JavaScript best practices.
*   Use functional components and hooks.
*   Write clean, readable, and well-documented code.
*   Use Tailwind CSS for all styling.

## Testing

*   All new features should be accompanied by corresponding tests.
*   Run `npm run test` to execute the test suite.

## Commits

*   Follow the conventional commit format.
